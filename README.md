It provides **self-paced, engaging learning activities** across four nature-based categories: **Animals, Plants, Seasons and Weather**.

The platform follows **autism-friendly design principles** to ensure a calm and supportive learning experience:

- Soft pastel colors and calm animations to reduce sensory overload  
- No timers or harsh error sounds  
- Positive reinforcement using gentle animations instead of red/error feedback  
- Touch-based interactive activities to support sensory and cognitive development  
- One idea per screen to minimize cognitive load
